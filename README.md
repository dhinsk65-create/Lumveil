# Lumveil ver. 1.7

MPVベースの動画プレイヤー。自動画質調整・フレーム補間・GPUシェーダー設定を搭載。

**制作:** ふぁん

---

## 特徴

### フレーム補間（デフォルトON）
30fpsの動画を60fps相当に補間し、なめらかに再生します。多くの動画プレイヤー（VLC等）では搭載していない機能で、特に映画やアニメで違いを実感できます。再生速度を変更した際（0.75x・1.25x等）のカクカク感も同時に軽減します。

### 自動画質調整（ver.1.7で刷新）
暗いシーンを検出し、暗部だけを持ち上げる「シャドウリフト」で見やすくします（中間調・ハイライトは保護されるため画面全体が白っぽくなりません）。強度は⚡AUTOボタンから「控えめ／標準／見やすさ優先／極暗」の4段階＋OFFを選択でき、意図的な暗転・ホラー演出は自動的に補正を弱めます。黒つぶれが激しい動画向けの「極暗」モードでは、輝度・コントラストも大きめに補正します。

### 映像モード
標準・シネマ・ダイナミック・鮮やか、TVのような映像プリセットをワンクリックで切り替えられます。

### 用途別画質プリセット
設定画面から「軽快」「標準」「アニメ高画質」「暗所優先」を選べます。画質・負荷・暗所補正の使いやすさをまとめて切り替えられ、詳細設定を知らなくても用途に合わせられます。利用者が追加した外部GLSLシェーダーは保持されます。

### Anime4Kプリセット
公式推奨の組み合わせ（モードA/B/C・軽量）をワンクリックで適用。ファイルを個別に選ぶ必要はありません。

### 字幕・音声トラック / スクリーンショット / ブックマーク / 履歴
複数トラックの切替、外部字幕の追加、静止画保存、任意位置のブックマーク、直近10件の再生履歴に対応。

### GPUシェーダー設定
スケール・クロマスケール・デバンディング・アンチリンギング・ディザリング・トーンマッピング・デインターレース等、映像品質に影響するGPU設定をGUI上で変更・保存できます。

### 全画面時の操作バー自動表示
全画面表示中は操作バーを非表示にし、映像を画面いっぱいに使います。マウスを画面下端に近づけるか右クリックすると一時的に表示されます。

### 続きから再生（ver.1.4で追加）
動画を閉じた位置を自動記憶し、次回同じファイルを開いたときに続きから再生します（先頭・末尾付近は対象外）。

### 連続再生・プレイリスト（ver.1.4で追加）
動画を1本開くと同じフォルダ内の動画が自動でプレイリスト化され、再生が終わると次のファイルへ自動的に進みます。複数ファイルをまとめてドラッグ＆ドロップした場合はその順番で再生します。

### 再生リストの表示
☷ボタンから現在の再生順を確認できます。再生したい動画を選ぶと、その動画へすぐ移動できます。

### A-Bリピート（ver.1.4で追加）
区間を指定してループ再生できます。ダンス・演出など繰り返し確認したい場面に便利です。

### 常に手前に表示（ver.1.4で追加）
他の作業をしながら「ながら見」したいときに、ウィンドウを常に最前面に固定できます。

### 字幕・音声の同期／見た目調整（ver.1.4で追加）
字幕の表示タイミング・サイズ、音声の再生タイミングをスライダーで細かく調整できます（Bluetoothイヤホン等での音ズレ補正にも）。

### AMD AMF補間（ver.1.5で追加）
AMD GPU内蔵の専用ハードウェアで、動きを解析して本物の中間フレームを生成します（フレームレートを約2倍に）。通常のフレーム補間より高品質ですが、AMD GPU + GPU再生支援（オン）が必要です。非対応環境や高フレームレート素材では自動的にスキップされ、通常のフレーム補間と同時使用はできません（排他制御）。

### 操作バーのカスタマイズ（ver.1.5で追加）
「設定 → 操作バー」タブで、常時表示するボタンをON/OFFしたり、ドラッグで並び順を変更できます。ウィンドウ幅が不足すると、右側のボタンから自動的に `⋯` メニューへ退避するため、ウィンドウを小さくしてもボタンが隠れて操作不能になりません。

### 音量・再生速度のUI刷新（ver.1.5で追加）
音量ボタンをクリックすると縦型スライダーがポップアップ表示されます（右クリックでミュート）。再生速度は数値ボタンをクリックして一覧から選ぶ方式に変わりました。

### 設定画面の統合（ver.1.5で追加）
「設定」「GPU」の2つに分かれていたウィンドウを1つに統合し、画質／なめらかさ／GPU再生支援／シェーダー／映像出力／再生／操作バーのタブに整理しました。

### 再生終了時・再開・並び順の設定（ver.1.5で追加）
「設定 → 再生」タブで、再生終了時の動作（停止／次の動画を再生）、前回位置からの再開有無、フォルダ末尾での動作（停止／先頭へ戻る）、連続再生時の並び順（ファイル名順／更新日時順）を選べます。

### 設定の自動保存
音量・画像調整・GPU設定・ブックマーク・履歴・スクリーンショット保存先・続きから再生の位置はアプリ終了時に自動保存され、次回起動時に復元されます。

## ファイル関連付け

動画ファイルをダブルクリックしたときにLumveil で開くよう設定できます。

1. `Lumveil_Associate.exe` をダブルクリック
2. 「このアプリがデバイスに変更を加えることを許可しますか？」→ **はい**
3. 関連付けたい拡張子にチェックを入れて「関連付ける」をクリック
4. 動画ファイルをダブルクリック →「アプリを選択」画面で **Lumveil.exe** を選び **「常に使う」** をクリック

> Windowsの仕様により、最終的な既定アプリの選択はユーザー操作が必要です。関連付けツールを実行することで選択候補の一覧にLumveil が表示されるようになります。
> 関連付けは一度設定すれば以降は不要です。解除したい場合は同じツールから「選択を解除」をクリックしてください。

## 設定画面（🎛 設定）

ver.1.7では暗闇補正をシャドウリフト方式に刷新しました。設定はアプリ終了後も保存されます。

### 画質タブ
明るさ・色味・字幕と音声の見た目を調整します。

- **用途別プリセット**: 「軽快」はAnime4Kと通常フレーム補間を停止して負荷を抑えます。「標準」は普段使い向けの画質と軽快さのバランス設定です。「アニメ高画質」はAnime4KモードAとデバンディングを適用します。「暗所優先」はAUTOの選択モードを標準にしますが、動画を開くまでAUTOを自動開始しません。プリセット対象を手動変更すると表示は「カスタム」に戻ります。
- **暗闇補正（自動、⚡ AUTO）**: 暗いシーンを検出して暗部だけを持ち上げるシャドウリフトを主体に補正（コントラスト・彩度は控えめな補助）。操作バーの`⚡ AUTO`ボタンをクリックすると「OFF／控えめ／標準／見やすさ優先／極暗」を選べるメニューが開きます。ボタンの文字色で現在のモードが分かります（水色＝控えめ、緑＝標準、黄＝見やすさ優先、紫＝極暗）。動画を開いてモードを選ぶと、ベースライン解析（数秒）後に自動調整が始まります。「暗闇補正の閾値」で発火の敏感さを調整（右＝敏感、左＝鈍感）
- **シャドウリフト（手動）**: AUTOを使わずに暗部の持ち上げ量を直接指定したいときのスライダー（AUTO停止中のみ操作可）
- **手動画像調整**: 輝度・実効コントラスト（-100〜+300）・ガンマ・彩度・色相を個別設定。実効コントラストの+100まではMPV標準処理、以降は明るさ方向を引き継ぐ拡張シェーダーで補います。映像モード（標準・シネマ・ダイナミック・鮮やか）のプリセットもここから
- **字幕・音声の同期調整**: 字幕遅延（±5秒）・字幕サイズ（0.5〜2.0倍）・音声遅延（±5秒）

### なめらかさタブ
- **フレーム補間**: 中間フレームを生成して映像をなめらかにする（デフォルトON）
- **AMD AMF補間**: AMD GPU専用の動き補償型フレーム補間（前述）。フレーム補間とは同時使用不可（排他制御）

### GPU再生支援タブ
GPUでデコードしCPU負荷を軽減する設定です。

| 選択肢 | 内容 |
|------|------|
| オン（推奨） | 実績のあるデコーダのみ使用 |
| オン（強制・実験的） | 対応していれば全て試行（不安定な場合あり） |
| オフ（CPUで処理） | ソフトウェアデコード。初期状態はこちら |

※ AMD AMF補間を使うには「オン」が必須です。

### シェーダータブ
- **Anime4Kプリセット**: 公式推奨の組み合わせをワンクリック適用（後述）
- **スケール**: 映像拡大アルゴリズム。Lanczos推奨
- **クロマスケール**: 色差成分の補間アルゴリズム。Spline36推奨
- **デバンディング**: バンディング（縞模様）除去。アニメや暗部のグラデーションに有効
- **アンチリンギング**: 拡大時の輪郭にじみ抑制
- **シグモイド拡大**: 拡大時のリンギング軽減
- **縮小補正**: 縮小時の暗部潰れ防止
- **GLSLシェーダー**: 外部シェーダーファイルを個別に追加（詳細・手動）

### 映像出力タブ
- **ディザリング**: 表示時の色深度変換で生じるバンディングを軽減（fruit推奨）
- **トーンマッピング**: 本物のHDR素材をSDR向けに変換（通常のSDR動画には無効）
- **デインターレース**: 古いTV放送由来のインターレース素材の横縞ノイズ除去

### 再生タブ
- **再生終了時**: 停止 / 次の動画を再生
- **前回位置から再開**: ON / OFF
- **フォルダ末尾**: 停止 / 先頭へ戻る
- **次の動画の並び**: ファイル名順 / 更新日時順

初期値はいずれも従来通りの挙動です。

### 操作バータブ
常時表示するボタンのON/OFF・並び替え（前述）に加えて、スクリーンショットの保存先フォルダをフルパス表示で変更できます。

## Anime4K GLSLシェーダー

`shaders\` フォルダにAnime4K（MIT License）のシェーダーが同梱されています。
アニメ動画の画質向上に効果的です。

### 適用方法（プリセット・推奨）

`🎛 設定` → シェーダータブ → Anime4Kプリセット から選ぶだけで、公式推奨の組み合わせが一括適用されます。

| プリセット | 用途 |
|-----------|------|
| なし | Anime4K系シェーダーを使わない |
| モードA | 迷ったらコレ（一般的なアニメ向け） |
| モードB | 元からぼやけている映像向け |
| モードC | 劣化が少ないきれいな映像・イラスト向け |
| 軽量 | PCが重いとき用（クリーンアップは省略） |

各ボタンはホバーで詳細説明が出ます。

### 個別に選びたい場合

「GLSLシェーダー（詳細・手動）」の `＋ 追加` から `shaders\` フォルダ内の任意のファイルを組み合わせられます。S / M / L / VL / UL の順に品質・GPU負荷が上がります。

> 実写動画にはAnime4Kは不向きです。シェーダータブの「全クリア」で無効化できます。

| ファイル | 説明 |
|---------|------|
| Lumveil.exe | アプリ本体 |

## 更新履歴

**ver.1.7**: 暗闇補正を全面刷新。全域コントラストに頼っていた旧方式から、暗部だけを持ち上げ中間調・ハイライトを保護する「シャドウリフト」主体の方式に変更しました。操作バーの`⚡ AUTO`ボタンから「控えめ／標準／見やすさ優先／極暗」の4段階を直接選べるようになり、意図的な暗転・ホラー演出では自動的に補正を弱めます。動画を切り替えたときに前の動画の基準値が残ってしまう不具合、霧・白基調シーンを暗所と誤判定する不具合、コントラスト・彩度が細かく上下振動する不具合、暗部の色が青白く飛ぶ不具合を修正。シャドウリフトの強度を直接指定できる手動スライダーを追加。

**ver.1.6**: 暗闇補正を再設計。実効コントラストを最大+300まで拡張し、+100以降もMPVの明るさ方向を引き継ぐGLSL補正へ変更。AUTOではコントラストに連動してガンマを最大-25、彩度を最大+60まで補正。手動時にも比率・補正・B/γ/C/Sを確認できるステータス表示を追加。AUTOとの併用でチラつく白飛び補正（ハイライト圧縮）は撤去。

**ver.1.5**: AMD AMF補間（GPUハードウェアによる動き補償型フレーム補間）を追加。操作バーのカスタマイズ（表示項目のON/OFF・並び替え・自動退避）に対応。音量ボタンをクリック式の縦スライダーに、再生速度をクリック選択式に変更。「設定」「GPU」ウインドウを1つに統合しタブ分け（画質/なめらかさ/GPU再生支援/シェーダー/映像出力/再生/操作バー）。再生終了時の動作・前回位置からの再開・フォルダ末尾の動作・連続再生の並び順を設定可能に。

**ver.1.4**: 続きから再生・連続再生（プレイリスト）・A-Bリピート・常に手前に表示・字幕/音声の同期調整を追加。シークバー操作とAUTO自動補正の負荷を軽減。「画像調整」ウインドウを「設定」に改称。

**ver.1.3**: 映像モード・Anime4Kプリセット・字幕/音声トラック・スクショ・ブックマーク・履歴・全画面操作バー自動非表示などを追加。細かいバグ修正多数。
