# Lumveil ver. 1.2

MPVベースの動画プレイヤー。自動画像調整・フレーム補間・GPUシェーダー設定を搭載。

**制作:** ふぁん × Claude Code

---

## 特徴

### フレーム補間（デフォルトON）
30fpsの動画を60fps相当に補間し、なめらかに再生します。多くの動画プレイヤー（VLC等）では搭載していない機能で、特に映画やアニメで違いを実感できます。再生速度を変更した際（0.75x・1.25x等）のカクカク感も同時に軽減します。

### 自動画像調整
暗いシーンを検出して自動的にガンマ・コントラスト・彩度を補正します。夜景や暗所シーンでも見やすい映像を維持します。

### GPUシェーダー設定
スケールアルゴリズム・デバンディング・アンチリンギング等、映像品質に影響するGPU設定をGUI上で変更・保存できます。

### 設定の自動保存
音量・画像調整・GPU設定はアプリ終了時に自動保存され、次回起動時に復元されます。

## 起動方法

`Lumveil.exe` をダブルクリックしてください。

---

## ファイル関連付け

動画ファイルをダブルクリックしたときにLumveil で開くよう設定できます。

1. `Lumveil_Associate.exe` をダブルクリック
2. 「このアプリがデバイスに変更を加えることを許可しますか？」→ **はい**
3. 関連付けたい拡張子にチェックを入れて「関連付ける」をクリック
4. 動画ファイルをダブルクリック →「アプリを選択」画面で **Lumveil.exe** を選び **「常に使う」** をクリック

> Windowsの仕様により、最終的な既定アプリの選択はユーザー操作が必要です。関連付けツールを実行することで選択候補の一覧にLumveil が表示されるようになります。
> 関連付けは一度設定すれば以降は不要です。解除したい場合は同じツールから「選択を解除」をクリックしてください。

| 操作 | 方法 |
|------|------|
| ファイルを開く | ⊕ボタン、またはウィンドウに動画ファイルをドラッグ＆ドロップ |
| 再生 / 一時停止 | ▶ボタン、スペースキー、または動画エリアをシングルクリック |
| 全画面 / 解除 | ⛶ボタン、F11キー、または動画エリアをダブルクリック |

## 自動画像調整（⚡ AUTO）

暗いシーンを検出して自動的にガンマ・コントラスト・彩度を補正します。

1. 動画を開く
2. `⚡ AUTO` ボタンをクリック
3. ベースライン解析（数秒）後、自動調整が開始されます

手動で調整したい場合は `🎛` ボタンから輝度・コントラスト・ガンマ・彩度・色相を個別に設定できます。

---

## GPU設定（⚙ GPU）

映像品質に影響するMPVのGPU設定を変更できます。設定はアプリ終了後も保存されます。

| 設定 | 説明 |
|------|------|
| スケール | 映像拡大アルゴリズム。Lanczos推奨 |
| クロマスケール | 色差成分の補間アルゴリズム。Spline36推奨 |
| デバンディング | バンディング（縞模様）除去。アニメや暗部のグラデーションに有効 |
| アンチリンギング | 拡大時の輪郭にじみ抑制 |
| シグモイド拡大 | 拡大時のリンギング軽減 |
| 縮小補正 | 縮小時の暗部潰れ防止 |
| フレーム補間 | 中間フレームを生成して映像をなめらかにする（デフォルトON） |
| GLSLシェーダー | 外部シェーダーファイルを適用（後述） |
| ハードウェアデコード | GPUデコードでCPU負荷を軽減 |

## Anime4K GLSLシェーダー

`shaders\` フォルダにAnime4K（MIT License）のシェーダーが同梱されています。
アニメ動画の画質向上に効果的です。

### 適用方法

1. `⚙ GPU` ボタンをクリック
2. GLSLシェーダーの `＋ 追加` をクリック
3. `shaders\` フォルダから目的のファイルを選択

### おすすめの組み合わせ

**バランス重視（多くの環境で動作）**
```
Anime4K_Clamp_Highlights.glsl
Anime4K_Restore_CNN_M.glsl
Anime4K_Upscale_CNN_x2_M.glsl
Anime4K_AutoDownscalePre_x2.glsl
Anime4K_AutoDownscalePre_x4.glsl
```

**軽量（GPU非力なPCに）**
```
Anime4K_Clamp_Highlights.glsl
Anime4K_Restore_CNN_S.glsl
Anime4K_Upscale_CNN_x2_S.glsl
Anime4K_AutoDownscalePre_x2.glsl
Anime4K_AutoDownscalePre_x4.glsl
```

**高品質（高性能GPU向け）**
```
Anime4K_Clamp_Highlights.glsl
Anime4K_Restore_CNN_L.glsl
Anime4K_Upscale_CNN_x2_L.glsl
Anime4K_AutoDownscalePre_x2.glsl
Anime4K_AutoDownscalePre_x4.glsl
```

> S / M / L / VL / UL の順に品質・GPU負荷が上がります。
> 実写動画にはAnime4Kは不向きです。GPUウィンドウの「全クリア」で無効化できます。

## 同梱ファイル

| ファイル | 説明 |
|---------|------|
| Lumveil.exe | アプリ本体 |
| Lumveil_Associate.exe | ファイル関連付けツール |
| libmpv-2.dll | MPV動画再生エンジン |
| ffmpeg.exe | サムネイルプレビュー用 |
| shaders\ | Anime4K GLSLシェーダー群 |
| _internal\ | Pythonランタイム（変更不要） |

設定ファイルは `%APPDATA%\Lumveil\` に自動作成されます。
